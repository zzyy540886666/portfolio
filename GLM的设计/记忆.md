### 设计规范
- **风格**：暖色暗调编辑式档案（warm-dark editorial archive）
- **字体栈**：Instrument Serif（展示）+ Newsreader（正文）+ JetBrains Mono（技术信息）
- **类别色**：网站=terracotta `#E07A4E`，软件=gold `#D4A95C`，小程序=sage `#8FA68E`
- **卡片栅格**：≥1020px 三列 / ≥1500px 四列（一行两个太大已废弃）
- **卡片视觉**：SVG 线稿 logo（每项目独立设计的 line-art 标记）+ 大号水印数字
- **详情布局**：横向 2 列分屏（左信息 / 右 carousel），整页锁定不滚动
  - `.detail-grid` 使用 `grid-template-columns:1fr 1fr`
  - `.detail-panel` 使用 `overflow:hidden` 避免整页滚动
  - `.detail-left` 内部可纵向滚动作为兜底
- **图片路径**：`const P = "../docs/";` — 文件位于子目录，需向上一级
- **路径安全**：`<img src="${P}${encodeURI(src)}">` 处理中文/空格文件名

### 关键技术决策
- **性能优化**（曾出现严重卡顿，已修复）：
  - 移除 `backdrop-filter: blur()` 应用于 sticky 元素
  - 移除 `mix-blend-mode: difference` 自定义光标 + rAF 循环
  - 降低 glow 的 `filter:blur()` 从 120px 到 80px
  - 卡片使用 `content-visibility:auto` + `contain-intrinsic-size`
  - 滚动事件 rAF 防抖
- **Coverflow 3D 轮播**：`perspective:1400px` + `translateZ` + `rotateY` + `scale` + `opacity` 分层
- **触摸/滚轮支持**：`pointerdown/move/up` + 横向 `wheel` 事件
