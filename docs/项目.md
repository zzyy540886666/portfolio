1.东软学生心理档案系统(商单)
- 概述：是面向学校心理健康管理的Web平台，用于学生心理档案管理、危机预警与干预追踪。
- 技术架构 ：采用 Vue3 + TypeScript + Vite 技术栈，基于 Element Plus 组件库与 Vuex 状态管理构建 SPA 架构，使用 ECharts 实现数据可视化、 WangEditor 支持富文本记录、 XLSX 实现批量导入导出。
- 核心功能：系统涵盖学生信息管理、心理咨询与访谈记录、问卷测评、预警分级审核（待审/批量复审/在库/等级变更）、危机与病历档案及统计看板等模块。
- 解决问题 ：解决了传统纸质档案分散、心理数据难以聚合分析、危机学生跟踪不及时的问题，实现心理健康数据数字化归档与预警闭环管理。

2.三角洲俱乐部电竞网站(商单)

3.宇树机器狗监控平台(商单)
- 概述：面向机器狗集群的可视化监控与智能巡检管理 Web 平台。

4.角膜地形图AI智能诊断系统(竞赛)
- 概述：角膜地形图AI智能诊断系统是一款眼科医疗AI应用网站，用于角膜地形图自动分类与圆锥角膜筛查。
- 技术架构 ：技术架构上采用PyTorch深度学习框架，基于ConvNeXt V2、MaxViT、Swin Transformer V2三模型加权集成，结合迁移学习、小样本学习与Grad-CAM++可解释性分析，前端以Streamlit+FastAPI提供服务并通过Docker容器化部署。
- 核心功能：系统功能涵盖图像上传预测、热力图可视化、临床指标分析与结构化报告生成。
- 解决问题 ：该项目解决了传统人工判读效率低、基层误诊率高、AI诊断黑盒不可信及医疗数据标注成本高的问题，500例数据即达92-96%准确率。
